AI Powered Fake News Detector

Developed a full-stack AI-powered application to detect fake news articles by analyzing text content for credibility. Key Achievements: Built an AI model using BERT for fake news classification. Integrated a Flask backend with a React frontend for seamless user interaction. Designed and managed a PostgreSQL database for storing predictions. Created dynamic Tableau dashboards for business intelligence and insights. Utilized Google Colab for model training with GPU resources. Tools and Technologies: AI/ML: Hugging Face Transformers, PyTorch Backend: Flask, SQLAlchemy Frontend: React, Axios Database: PostgreSQL, pgAdmin BI: Tableau for data visualization

Chatbot with Amazon Lex

A BankerBot chatbot leveraging Amazon Lex to handle banking-related queries such as balance inquiries, enhancing user experience with natural language processing. Key Achievements: Configured intents (e.g., CheckBalance, WelcomeIntent) and slots for personalized interactions. Integrated AWS Lambda to generate dynamic, real-time responses for bank balance inquiries. Designed fallback mechanisms with user-friendly messages for unrecognized inputs. Reduced repetitive prompts by enabling Lex to remember user details like account type. Tools and Technologies: AWS Lex: For creating the chatbot and managing intents. AWS Lambda: To generate real-time responses and implement custom logic. Key Features: Dynamic balance inquiries powered by Lambda. User verification through custom slots (e.g., account type, date of birth). Multiple fallback variations for natural and engaging conversations. Key Learnings: Seamless integration of Amazon Lex with Lambda for dynamic interactions. Optimized chatbot responses with a confidence threshold of 0.40 for better accuracy. Gained experience in building conversational interfaces for real-world applications.
